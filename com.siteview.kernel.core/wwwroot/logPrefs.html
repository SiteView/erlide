Log File Preferences

Sustained operations monitoring of web systems can generate a large amount of data. SiteView stores this data in log files on the SiteView machine. The following settings allow you to limit how much data is saved in SiteView log files.

[[[page:print_settings()]]]

Database Logging

[[[topmenu:productname()]]] can optionally log a copy of all the monitoring data into a database. Any database which supports the JDBC standards can be used, including Microsoft SQL Server and Oracle.
(Note: changes to Database Logging preferences do not take effect until [[[topmenu:productname()]]] is restarted)
