Description

Port Monitor.

Description: Using gen_tcp,gen_udp erlang module for sokect programming, Verifies that a connection can be made to a network port and measures the length of time it takes to make the connection. Optionally, it can look for a string of text to be returned or send a string of text once the connection is made.
